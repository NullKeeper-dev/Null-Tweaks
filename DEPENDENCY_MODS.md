# Null Tweaks Dependency Mods

Install these dependency mods alongside the Null Tweaks jar that matches your
Minecraft version.

## Required Mods

- Fabric Loader
- Fabric API
- YetAnotherConfigLib
- Mod Menu

## Matching Versions

| Minecraft | Fabric Loader | Fabric API | YetAnotherConfigLib | Mod Menu |
| --- | --- | --- | --- | --- |
| 26.1 | 0.19.3 or newer | 0.145.1+26.1 or newer | 3.9.5 or newer | 18.0.0-beta.1 or newer |
| 26.1.1 | 0.19.3 or newer | 0.145.4+26.1.1 or newer | 3.9.5 or newer | 18.0.0-beta.1 or newer |
| 26.1.2 | 0.19.3 or newer | 0.154.2+26.1.2 or newer | 3.9.5 or newer | 18.0.0-beta.1 or newer |
| 26.2 | 0.19.3 or newer | 0.154.2+26.2 or newer | 3.9.5 or newer | 20.0.0-beta.4 or newer |

Only these dependency mods are required for Null Tweaks.
